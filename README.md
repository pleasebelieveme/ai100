# 🎬 콘티·대본·음악까지 한 번에: AI 숏폼 콘텐츠 어시스턴트

## 📌 개요
유튜브 영상 2개를 입력하면, 자막·메타데이터를 자동 수집해  
**영상별 상세 분석 → 종합 주제 추천 → 맞춤 대본 생성 → 3초 단위 콘티 → 음악 추천**  
까지 한 번에 만들어주는 **Streamlit 기반 웹 앱**입니다.
